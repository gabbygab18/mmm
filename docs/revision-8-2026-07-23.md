# Revision 8 — 23 July 2026 (mobile asset pack)

## What arrived

The mobile pack ships **phone-specific artwork already composed at the phone
band ratio** — which is exactly what was missing. Unlike the desktop pack, these
aren't flattened desktop layouts that need decomposing; each one is drawn for
the narrow band.

Measured from the artwork's own opaque area:

| Asset | Opaque size | Ratio | Photo |
|---|---|---|---|
| About header | 1100 × 650 | 1.69 | rows 0–617, right 43% |
| FAQ background | 1100 × 387 | 2.84 | none — gradient + wave only |
| Contact background | 1100 × 387 | 2.84 | none |
| How It Works header | 1100 × 279 | 3.94 | spans nearly full width |

The FAQ mobile banner having no photograph confirms the mobile mockup — the
gentleman only appears on desktop.

## Wired in

`PageHero` gained a `mobileImage` prop. Below `sm` it renders that banner
full-bleed and suppresses the layered composition (cutout photo, notes, and the
separate wave, since the mobile banners carry their own). From `sm` up, the
desktop layer composition is unchanged.

- **About** — mobile banner plus the mobile portrait of Margaret, swapped via
  `<picture>` at the 639px breakpoint.
- **FAQ / Contact** — mobile background banners.
- **Contact** — mobile Palm Beach map.
- **How It Works** — the two faded side crops are hidden on phones in favour of
  the mobile header banner.

Extracted to `public/mmm/mobile/` by `scripts/extract-mobile.py`, 16 files,
~5.5 MB. Same rule as before: only real artwork, never copy-as-image.

## One fix this surfaced

`minHeight` was `min-h-[300px]`, which overrode the aspect ratio on phones — a
390px-wide About hero came out 300px tall (ratio 1.30) instead of the artwork's
1.69. Lowered to `min-h-[190px]` so the artwork's own ratio governs.

Verified at 390px:

| Page | Band | Ratio | Target | Headline fits |
|---|---|---|---|---|
| About | 390 × 231 | 1.69 | 1.69 | yes |
| FAQ | 390 × 195 | 2.00 | 2.00 | yes |
| Contact | 390 × 190 | 2.05 | 2.20 | yes |

Contact is floored by the 190px minimum rather than hitting 2.20 exactly; that
is deliberate, since a 177px band leaves the two-line subtitle nothing to sit in.

No horizontal overflow at 390 / 768 / 1024 / 1440 across About, FAQ, Why Music
Matters, Contact, How It Works, Facility Registration, Home and Get Started.
