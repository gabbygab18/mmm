# Website Project Scope (Client-Friendly)

Date: June 3, 2026

This document explains what is included in the current project, what is not included, and what can be added later.

## Included (MVP Scope)

- A complete website platform that connects musicians and memory care facilities.
- Education page on what the site is, info on residents, dos & donts, sample playlists, etc.
- Separate account types for musicians, facilities, and admin.
- Sign-up, login, and account setup for each role.
- Musician profiles and facility profiles.
- Local matching based on ZIP code distance.
- Ability for both sides to post availability and find matches.
- Ability for either side to send a booking request.
- Request tracking (sent, accepted, completed, cancelled) with history.
- A schedule view for accepted and completed events.
- In-app notifications and alerts.
- Email notifications and alerts.
- Basic legal pages for launch (Privacy Policy and Terms of Service).
- A simple process for privacy/data requests (example: a user can request a copy of their account data or request account deletion).
- Admin controls to review accounts, manage issues, and moderate content.
- Video/media workflow with review and publish controls (uploaded by musician, approved by admin).
- Privacy defaults on public profiles (limited public name display and email-first contact).
- Volunteer model at launch (no payments built in at this stage).
- Adults-only at launch (18+).

## Included (Mark's Team)
- Design of logo
- Color scheme
- Look & Feel / Navigation Design
- Testing (User experience and functional bugs)


## Not Included in This Project

- Payment processing.
- Mobile apps (iPhone/Android apps).
- Real-time chat or direct messaging.
- Text message (SMS) notifications.
- Advanced analytics dashboards.
- Automated ranking/recommendation engine.

## Deferred for Future Development

Either planned for a later phase or things to consider:

- Background-check integration with an outside provider.
- Advanced media pipeline (including centralized MMM auto-posting workflow).
- Musician education quiz/comprehension check.
- Support for facilities with more than one location.
- Ratings and review system.
- Marketing emails (newsletter-style) and a full email preference center/unsubscribe management.


## How to Use This in the Proposal

- Everything in Included Now is part of the current delivery.
- Everything in Not Included is excluded from this project price.
- Everything in Deferred can be quoted as optional Phase 2 work.
- Email sending is included in MVP scope and is part of launch delivery.
